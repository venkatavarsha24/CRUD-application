Product Management Application (MERN Stack):
Project Overview:
SAC (Smart Accessories Collection) is a full-stack Product Management Application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application is designed specifically for managing different types of bags such as school bags, travel bags, handbags, laptop bags, trolley bags, backpacks, hiking bags, and duffel bags.
The primary objective of this project is to demonstrate the complete CRUD (Create, Read, Update, Delete) workflow by integrating a React frontend with a Node.js and Express backend, while storing and managing product information in MongoDB.
